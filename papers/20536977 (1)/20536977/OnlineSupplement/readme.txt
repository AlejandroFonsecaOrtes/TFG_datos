This directory contains the online supplements (a PDF file, datasets and R code).

## Datasets
The datasets used in this paper can be loaded from the following link:
http://www.timeseriesclassification.com/description.php?Dataset=Heartbeat
The R script **data_pre** in the folder **DATA**  contains the code to process the data

## DPCA code
The folder **R CODE** contains the R function files used in numerical studies of this paper
The R script **main_common** contains R code to implement experiments under the common design in Section 4
The R script **main_irregular**  contains R code to implement experiments under the irregular design in Section 4
The R script **Heartbeat_common** contains R code for data application under the common design in Section 5
The R script **Heartbeat_irregular** contains R code for data application under the irregular design in Section 5
